# 바근지팡

9살 아이가 말한 아이디어를 바탕으로 만든 간단한 MVP 게임입니다.

## 지금 버전

- 주인공 캐릭터들은 밑에 대기합니다.
- 이번에 쏠 주인공이 밑 가운데 발사 자리로 동글동글 굴러옵니다.
- 밑 가운데 주인공을 누르고 당긴 뒤 놓으면 위쪽으로 발사됩니다.
- 위쪽에는 같은 캐릭터, 다른 캐릭터, 똥, 폭탄이 섞여 나옵니다.
- 주인공과 같은 캐릭터를 맞히면 단계 성공입니다.
- 다른 캐릭터를 맞히거나 빗나가면 다시 제자리로 돌아옵니다.
- 똥을 맞히면 성공이 아니고 다시 돌아옵니다.
- 폭탄을 맞히면 실패입니다.
- 4단계부터는 위쪽 캐릭터와 장애물이 움직입니다.
- 단계가 올라갈수록 상단이 더 빽빽해지고 목표가 작아집니다.
- 조준 점선은 벽에 튕기는 예상 경로를 보여줍니다.

## 실행

`index.html` 파일을 브라우저로 열면 바로 실행됩니다.

## 조카에게 공유하기

GitHub Pages로 무료 공유할 수 있습니다.

1. GitHub에서 새 Public 저장소를 만듭니다.
2. 이 폴더의 파일들을 저장소에 올립니다.
3. 저장소의 Settings > Pages에서 Branch를 `main` 또는 `master`, 폴더를 `/root`로 선택합니다.
4. 저장 후 생기는 Pages 링크를 조카에게 보내면 됩니다.

수정할 때는 이 폴더에서 파일을 고친 뒤 GitHub에 다시 올리면 같은 링크가 최신 게임으로 바뀝니다.

## 안드로이드 앱으로 만들기

Expo 없이도 가능합니다. 쉬운 순서는 아래와 같습니다.

1. 지금 웹 MVP로 조작감과 룰을 먼저 확인합니다.
2. 반응이 좋으면 Android Studio 프로젝트를 만듭니다.
3. 이 게임 파일들을 앱 안에 넣고 Kotlin WebView로 감싸 APK/AAB를 빌드합니다.

이 방식은 게임 로직과 캐릭터 이미지를 거의 그대로 재사용할 수 있어서 처음부터 네이티브 게임을 만드는 것보다 부담이 적습니다.
